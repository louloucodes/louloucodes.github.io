--- layout: default title: Blog navigation: true --- {% comment %} =============== From: https://codinfox.github.io/dev/2015/03/06/use-tags-and-categories-in-your-jekyll-based-github-pages/ The following extracts and sorts all the tags from all the posts. =============== {% endcomment %} {% assign rawtags = "" %} {% for post in site.posts %} {% assign ttags = post.tags | join:'|' | append:'|' %} {% assign rawtags = rawtags | append:ttags %} {% endfor %} {% assign rawtags = rawtags | split:'|' | sort %} {% comment %} ================ The following removes duplicated and invalid tags. ================ {% endcomment %} {% assign tags = "" %} {% for tag in rawtags %} {% if tag != "" %} {% if tags == "" %} {% assign tags = tag | split:'|' %} {% endif %} {% unless tags contains tag %} {% assign tags = tags | join:'|' | append:'|' | append:tag | split:'|' %} {% endunless %} {% endif %} {% endfor %}
